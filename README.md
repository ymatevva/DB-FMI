# DataBases  
###### Задания към курса по Бази данни към ФМИ, СУ, 2024/2025 година, летен семестър, специалност Информационни системи

### Упражнения  
- Извличане на данни от база данни
    - [Упражнение 1](<./Lab_01/>) – Работа с `SELECT` заявки  
      Основи на извличането на данни от една таблица.
      
    - [Упражнение 2](<./Lab_02/>) – Съединения (`JOIN`)  
      Работа с множество таблици чрез различни видове съединения: `INNER`, `LEFT`, `RIGHT`, `FULL`.

    - [Упражнение 3](<./Lab_03/>) – Подзаявки (`Subqueries`)  
      Използване на вложени заявки в `SELECT`, `WHERE`, и `FROM`.

    - [Упражнение 4](<./Lab_04/>) – SQL Функции  
      Вградени функции за текст, дати и числови стойности (`UPPER`, `ROUND`, `NOW`, и др.).

    - [Упражнение 5](<./Lab_05/>) – Групиране (`GROUP BY`)  
      Агрегиране на данни чрез `GROUP BY`, `HAVING`, и агрегатни функции (`COUNT`, `AVG`, `SUM`, и др.).

    - [Упражнение 6](<./Lab_06/>) – Обобщение и практика  
      Смесване на различни концепции чрез комбинирани задачи.
